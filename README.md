- **Extract contact info:** Gather addresses, emails, phone numbers, and other public details from Facebook pages.

## What Data This Scraper Extracts

| Field Name        | Field Description                                                               |
|-------------------|---------------------------------------------------------------------------------|
| address           | The physical address listed on the Facebook page.                               |
| email             | The email address found on the Facebook page.                                   |
| phone             | The phone number associated with the Facebook page.                             |
| likes             | The number of likes the page has received.                                     |
| website           | The website URL linked on the Facebook page.                                    |
| check-ins         | The number of check-ins associated with the page.                              |
